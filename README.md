## About this project

This project is made for studying and learning about meilisearch engine using laravel 10 and laravel scout. Laravel sail was used to configure the project.

## How to install

Run for alias when using sail command in terminal:
- alias sail='[ -f sail ] && sh sail || sh vendor/bin/sail'

- sail up -d
- sail artisan migrate --seed
- sail artisan scount:import "App\Models\Post"
- sail artisan scout:sync-index-settings
